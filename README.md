# SnapToStory 프로젝트

이 프로젝트의 목적과 주요 기능을 여기에 기술합니다.

## 폴더 구조
- `primary_data`: 원본 데이터
- `secondary_data`: 전처리 데이터
- `intermediate_results`: 중간 분석 결과
- `visualizations`: 시각화 이미지
- `interim_reports`: 중간 보고서
- `scripts`: 분석 스크립트
