# SnapToStory 프로젝트

10장의 사진 중 최고의 순간을 선별하여 이야기를 입히는 AI 기반 스토리 생성 서비스입니다.

## 기술 스택
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide-react
- **Animation**: Framer Motion

## 폴더 구조
- `src/components`: UI 컴포넌트 (Layout, Sections)
- `src/app`: Next.js App Router (Pages, Styles)
- `primary_data`: 원본 데이터
- `secondary_data`: 전처리 데이터
- `interim_reports`: 단계별 중간 보고서
- `scripts`: 분석 및 유틸리티 스크립트

## 시작하기
```bash
npm install
npm run dev
```
